The heavyweight: platform choice, secure design, and data access patterns.

  • Storage & schema choice: AlloyDB, Spanner, Bigtable, Firestore; signed URLs

💡 Coach's Study Tip!

This exam rewards hands-on familiarity, so spin up a Cloud Run service from source and wire a Pub/Sub/Eventarc trigger before exam day. Lock down the auth ladder (Application Default Credentials → service accounts → Workload Identity Federation — never long-lived keys) and the retry pattern (exponential backoff with jitter). When a question stresses least operational overhead, the answer is almost always the most managed option (Cloud Run, signed URLs, Secret Manager).
