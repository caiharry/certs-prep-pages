💡 Coach's Study Tip!

Two anchors carry this exam: the Vertex AI service map (Datasets, Feature Store, Workbench, Training, Pipelines, Prediction, Model Monitoring) and the build-vs-buy ladder (BigQuery ML → AutoML → custom training; ML API/Model Garden for pretrained). For monitoring questions, distinguish training-serving skew (preprocessing mismatch) from prediction/feature drift (input distribution changes over time). GenAI topics (Model Garden, Agent Builder, RAG, evaluation) are now fair game — don't skip them.
