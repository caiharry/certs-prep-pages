💡 Coach's Study Tip!

This exam is platform-specific: live in Google SecOps (SIEM + SOAR) and Security Command Center. Get fluent in YARA-L rule structure and the UDM (Unified Data Model) — detection engineering is the biggest slice. Know the tool split cold: SCC = posture/misconfig/CSPM & Event Threat Detection; SecOps SIEM = detections over ingested telemetry; SecOps SOAR = playbooks/case management; GTI = threat intel/IOCs. Silent source detection (a feed stopped sending logs) is a favourite observability discriminator.
